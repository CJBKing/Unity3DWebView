Unity3DWebView
==============

Unity3DWebView<br>
Unity3D移动网页浏览窗口插件。<br>
在游戏中或应用中可以用于打开某个公告页面或者活动页面。<br>
这个webview模块除了可以打开某个链接外，还可以在点击链接的时调用u3d中的接口。<br>
u3d能调用访问网页，网页也能根据链接调用u3d，这样就能使得网页和U3D结合使用在游戏里了。<br>
